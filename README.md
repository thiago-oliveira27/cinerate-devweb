# CineRate

Plataforma colaborativa para avaliar e descobrir filmes, séries e documentários.

---

## Funcionalidades

### Visitante (sem login)

- Visualizar página inicial com informações sobre o sistema
- Consultar catálogo completo de títulos
- Ver detalhes de cada título (gênero, ano, nota média, avaliações)
- Buscar títulos por nome
- Filtrar por tipo: Filmes, Séries ou Documentários

### Usuário Registrado

- Criar conta, fazer login e logout
- Cadastrar novos filmes, séries e documentários
- Editar títulos próprios
- Avaliar títulos com nota de 1 a 5 estrelas e comentário
- Editar perfil (nome, e-mail, senha)
- Excluir a própria conta

### Administrador

- Todas as funcionalidades do usuário registrado
- Remover qualquer título do sistema
- Excluir comentários de qualquer usuário
- Listar e excluir usuários
